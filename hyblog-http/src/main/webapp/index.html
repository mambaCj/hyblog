呆萌！巨型仙人掌遭小鸟挖洞筑巢

2015-05-19 14:14 环球网

据英国《镜报》报道，在美国亚利桑那州的南部城市图森，一棵树形仙人掌，因有小鸟在其主茎上挖洞筑巢，看起来就像一张笑脸，而且其两旁的侧茎像是在向路人招手，整个造型十分友好可爱。

据悉，这是一种树形仙人掌，也是世界上最高的仙人掌，多产于墨西哥与美国加利福尼亚州。由于长在沙漠地区，许多鸟类也会为了生存在这种仙人掌上挖洞筑巢，于是就出现了这种可爱的仙人掌造型。
